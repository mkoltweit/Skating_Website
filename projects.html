JavaScript Used

Functions
Functions were utilized a few times in this project. I used named functions that were called to add event listeners to buttons and hover events for the menu. In addition to this, I also used anonymous functions as well to add event listeners to HTML elements on pages.You can view the page at the following link: Function Example
If, Else Statements
I used if statements in different ways. One helpful way I used if statements was to check if an element exists on the page. This was needed for some pages where I was trying to call an element that didn't exist when trying to add an event listener. I also used if/else statements for my schedule page to see which parameter was passed into a function so it knows which table needs to be created when a user clicks on a date on the calendar. You can view the page at the following link: If, Else Statement Example
For Loops
I used for loops on my Schedule page. Since the calendar needed click events, I thought it'd be a good use case for a for loop. So I used a for loop to add click event listeners to each of the list items and specified them by targeting them with their class names. You can view the page at the following link: For Loop Example
Events
I used quite a bit of event listeners on this site. They are very useful in adding interactivity to a page. I used click events to show schdules on the schedule page. I used hover events on the navigation as well. You can view the page at the following link: Events Example
jQuery
I used jQuery for the slideshow used on the index page. In the previous project, I thought it was really cool how a simple slideshow was made using such little code. It really showed me the benefits of using jQuery and how less code can be used to do more. You can view the page at the following link: jQuery Example
